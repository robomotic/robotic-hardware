# Torque Repository

## Overview

This repository is dedicated to **modeling robotic hardware**. It provides in-depth tutorials, interactive simulations, and practical guides for understanding the physics and mechanics behind robotic actuators, joint dynamics, and control systems.

## Purpose

Learn how to:
- Understand torque specifications and their role in robot design
- Model inertia and its effects on humanoid robot performance
- Calculate actuator requirements for robotic limbs
- Apply physics principles to real-world robotics problems
- Bridge the gap between theory and hardware implementation

## Contents

- **`notebooks/`** – Interactive Jupyter notebooks covering robotics fundamentals:
  - `torque_is_all_you_need.ipynb` – Introduction to torque, motors, control, and actuator sizing
  - `progressive_mujoco_biped_tutorial.ipynb` – Step-by-step MuJoCo tutorial from 1-DOF knee control to a planar 2D point-foot biped

## Getting Started

1. Install dependencies: `numpy`, `matplotlib`, `ipywidgets`, `mujoco`
2. Optional for MJX section: `jax`, `jaxlib`
3. Open notebooks in Jupyter or Google Colab
4. Run cells sequentially and interact with sliders to build intuition

## Target Audience

Beginners interested in robotics hardware design, actuator selection, and understanding the physics behind humanoid robots.
